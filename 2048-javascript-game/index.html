Welcome to 2048 JavaScript Game

No internet, no mobile data, just play game and make score

About Game

This is an number counting game, made with JavaScript. You just need to add number one after another and your aim is to make 2048.

How to play

Read the instruction carefully. You will get only 60 seconds to make 2048.When the game page will appear, from this time the countdown will be started. There will be nine cell each containing a number from 1 to 9.You can select any cell by using mouse or pressing any key from 1 to 9 from keyboard.When you select any cell it background will change and this time it also show you on which cell you can go for next step.

You can play the game without mouse. Just need to use the upper key, right key, left key and bottom key. If you want to play the game from beginning just press the Enter button.
